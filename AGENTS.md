# Repository Guidelines

Monorepo for the @biscuittin ESLint config, formatter plugin, and plain parser packages.

- Package manager: pnpm
- Node: >= v24.11.1 < 25
- Non-standard command: `pnpm run typegen` (generates types via `tsx scripts/typegen.ts`)
